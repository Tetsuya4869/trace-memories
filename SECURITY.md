# セキュリティポリシー

## 機密情報の管理

### API キー・トークン

| 情報 | 保管場所 | Git 管理 |
|------|---------|---------|
| Mapbox Public Token | `.env` | `.gitignore` で除外 |
| Mapbox Secret Token | `.env` / `~/.netrc` | `.gitignore` で除外 |
| iOS 署名情報 | Xcode / Keychain | リポジトリ外 |
| Android 署名キー | `key.properties` | `.gitignore` で除外 |

### ルール

- API キー・トークン・パスワードはソースコード内に直接記述しない
- `.env` ファイルは `.gitignore` に登録し、Git にコミットしない
- `pubspec.yaml` の `assets` セクションに `.env` を追加しない
- CI/CD 環境では環境変数またはシークレット管理サービスを使用する

### 対象ファイル一覧

以下のファイルは `.gitignore` で保護されています:

```
.env
.env.*
android/gradle.properties  (Secret Token を含む場合)
android/key.properties
ios/Runner/MapboxSecrets.plist
```

## ユーザーデータの取り扱い

### データの保存場所

すべてのユーザーデータは端末内にローカル保存されます。

| データ | 保存場所 | 外部送信 |
|-------|---------|---------|
| GPS 軌跡 | 端末メモリ (将来: SQLite) | なし |
| 写真メタデータ | 端末メモリ (将来: SQLite) | なし |
| サマリーテキスト | 端末メモリ (将来: SQLite) | なし |
| オンボーディング状態 | メモリ (将来: SharedPreferences) | なし |

### 第三者サービス

| サービス | 用途 | 送信データ |
|---------|------|----------|
| Mapbox | 地図タイル取得 | 地図の表示領域座標 |
| OpenStreetMap (Web 版) | 地図タイル取得 | 地図の表示領域座標 |

## 権限の管理

アプリが要求するシステム権限:

| 権限 | 用途 | 必須 |
|------|------|-----|
| 位置情報（使用中のみ） | GPS 軌跡の記録 | 推奨 |
| 写真ライブラリ（読み取り） | 位置情報付き写真の表示 | 推奨 |

- 権限が拒否された場合、バナーで案内を表示する
- アプリは権限なしでも起動可能（機能制限あり）
- 権限の再取得は端末の設定アプリから行う
